Service Map Visualization Kit

Interactive dependency graph for 150+ services

Status: Draft / POC

Target: GitHub Pages / MWA

How it works

The graph renders client-side from a static CSV registry (public/data/services.csv). Cytoscape.js + fcose handles the layout; PapaParse converts CSV rows into nodes and edges.

Drop an updated CSV in public/data/ and rebuild to publish a fresh “living diagram.”

Data contract

  • id, label, domain required
  • depends_on: semicolon-separated
  • tier: 1 (Critical) → 3 (Internal)
